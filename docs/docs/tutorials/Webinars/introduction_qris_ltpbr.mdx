In February of 2024, we started doing some introductory workshops on Riverscapes Apps and Tools for the Bureau of Land Management. The main audiences were BLM staff and partners working on the BLM Restoration Landscapes. Initial workshops were taught in Salt Lake City, Utah; Missoula, Montana; Cheyenne, Wyoming; and Twin Falls, Idaho.
