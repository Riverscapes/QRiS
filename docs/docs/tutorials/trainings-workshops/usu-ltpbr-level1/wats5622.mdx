## Module 3.1 - Reading Riverscapes 

### Overview

This module provides an introduction to the planning process of Low-Tech Process-Based Restoration of Riverscapes. Students will learn how to map riverscapes, assess current conditions and estimate recovery potential to support LTPBR projects.

## Module 3.2 - Risk Assessment, Condition Assessment, and Recovery Space 

### Overview

This module we focuses on identifying potential risks and negative impacts from restoration implementation and conducting a condition assessment focused on hydrologic, geomorphic, riparian, and biological attributes. Once the condition of a riverscape is determined we try to determine it's current trajectory (is it getting worse, stable, improving), and recovery potential (i.e., can we restore natural processes).
