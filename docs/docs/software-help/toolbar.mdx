QRiS functionality is accessed through the QRiS toolbar. From here you can create and open projects, as well as access the online help and QRiS Settings.

![toolbar](/images/software-help/toolbar.png)

### Hiding & Showing the QRiS Toolbar

The QRiS toolbar can be hidden and shown by right clicking on the gray area to the right of the QGIS menus and toolbars. Look for QRiS under the **Toolbars** subheading. You access the same list via the View -> Toolbars menu.

You may move the toolbar by clicking and dragging the handle located on the right side of the toolbar.

### Hiding & Showing the QRiS Project Explorer

Clicking the QRiS icon on the left side of the QRiS toolbar will hide and show the QRiS Project Explorer. The explorer is used to navigate and manage your QRiS projects. You can only ever have one QRiS project open at a time.

## Project Menu
The Project menu is accessed by clicking the **Project** button on the QRiS toolbar. This menu provides options for creating, opening, and managing QRiS projects.

![project menu](/images/software-help/project-menu.png)

### New QRiS Project

To create a new project, click the **Project** button and then **Create New Project** on the QRiS toolbar. This will open the **New QRiS Project** dialog. See [Creating a New Project](/software-help/projects/#creating-a-new-project) for more information.

### Open QRiS Project

To open an existing project, click the **Project** button and then **Open QRiS Project** on the QRiS toolbar. This will open the **Open QRiS Project** dialog. See [Opening an Existing Project](/software-help/opening-an-existing-project) for more information.

### Recent QRiS Projects

QRiS keeps a list of recently opened projects. To open a recent project, click the **Project** button and then select the project from the list. This will open the project.

### Close Project

To close the current project, click the **Project** button and then **Close Project** on the QRiS toolbar. This will close the project and return you to the QRiS Project Explorer.

## Help Menu
The Help menu is accessed by clicking the **Help** button on the QRiS toolbar. This menu provides options for accessing the online help documentation and the QRiS settings.

![help menu](/images/software-help/help-menu.png)

### Settings

The Settings menu is accessed by clicking the **Settings** under the Help Menu on the QRiS toolbar. This menu option will open the QRiS Settings dialog. 

#### General Settings
The General tab provides options for configuring general QRiS settings.

![general settings](/images/software-help/settings-general.png)

| Setting             | Description                                         |
|---------------------|-----------------------------------------------------|
| Default Export Path | Set the (optional) default root path for exporting projects. You can always specify a different path when exporting, this option just provides a shortcut. |
| Remove QRiS Project Map Layers on project close | Remove all QRiS project layers (except for Basemaps) from the current map when the project is closed.|
| Default Dock widget location | Set the default location for the QRiS dock widget when a project is opened. Options are docking to the left (default) or right side of the main QGIS window. |

#### Protocol Settings
The Protocols tab provides options for configuring the QRiS protocol settings.

![protocol settings](/images/software-help/settings-protocols.png)

| Setting | Description                                         |
|---------|-----------------------------------------------------|
| Local Custom Protocols Folder  | Set the folder for local custom QRiS protocols. |
| Show experimental protocols | Show experimental protocols when adding layers to a data capture event. |
